- First clone to vps machine

git clone https://github.com/KimelirR/givecloudTask.git givecloudc

- move file to /var/www/html 

sudo mv givecloud/ /var/www/html/givecloud

- For any edit do this 
cd /var/www/html/givecloud

nano src/config.php

- and paste below code 


- Give nginx permission to access givecloud files:
sudo chown -R www-data:www-data /var/www/html/givecloud

- Give read and write permissions
# File and directory permission you should give:
sudo find /var/www/html/givecloud/ -type d -exec chmod 755 {} \;
sudo find /var/www/html/givecloud/ -type f -exec chmod 644 {} \;

- Then restart nginx to take effect to changes
systemctl restart nginx
systemctl status nginx
